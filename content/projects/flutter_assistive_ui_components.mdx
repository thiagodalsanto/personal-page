A Flutter package of assistive components for interfaces used by people with fine motor difficulties. It provides larger, more touch-tolerant widgets that can be configured at runtime, with support for haptic feedback, visual feedback, activation delay, and accessibility presets. It provides larger, more touch-tolerant widgets that can be configured at runtime, with support for haptic feedback, visual feedback, activation delay, and accessibility presets.

Components:
- Input Components: Textfield, Checkbox, Radio, Slider, Switch, Dropdown, TimePicker;
- Navigation Components: TabBar, BottomBar;
- Interaction Components: Button, BigButton, TextButton, IconButton;
- Feedback and Multimodal Components: LinearProgressIndicator, AlertDialog.

Features: 
- Touch-tolerant UI components with configurable size, padding, and colors;
- Runtime configuration through UserComponentProvider;
- Optional haptic feedback for interactive elements;
- Optional floating visual feedback messages;
- Optional activation delay through pressDelay;
- Accessibility presets for fine motor, severe motor, and aphasia-oriented configurations;
- A draggable floating assistive button that opens the runtime configuration UI.
